Chantry Island

Client Site

Project Overview

For this client project I worked with Brandon Muise to re-design the Chantry Island site as well as build a new gallery and a custom Content Management System. We also implemented a custom Google Maps API with location search functionality.

Our goal with this particular project was to make the site as user-friendly and simple to navigate as possible. We achieved this by implementing responsive and legible fonts as well, large images for overall visual appeal and an overall engaging flow of content and information.

Technologies

  • HTML5
  • CSS3
  • jQuery, AJAX, JSON
  • Custom CMS
  • PHP
  • Foundation Framework
  • Fully Responsive
